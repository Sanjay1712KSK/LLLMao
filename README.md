# Project-Tony-Stank
A lightweight ChatGPT-style local AI assistant powered by Ollama with RAG, document indexing, workspace awareness, and live system monitoring.

## Supported Local Models

This application connects to locally installed Ollama models.

Recommended models:
- llama3:8b
- qwen2.5-coder:14b
- nomic-embed-text
- llava

Users are responsible for installing and managing their own local models through Ollama.
