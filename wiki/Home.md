## Key Features

- **Offline-First**: Works entirely locally without internet
- **Privacy-Focused**: All data stays on your machine
- **Semantic RAG**: Upload documents and search semantically
- **Workspace Indexing**: Analyze entire codebases intelligently
- **Multimodal**: Understand images and screenshots
- **Audio**: Speech-to-text and text-to-speech support
- **Developer Tools**: Git integration and terminal access
- **System Monitoring**: Real-time GPU/CPU metrics

## Core Principles

1. **Offline First** - No cloud dependencies
2. **Privacy Focused** - Your data stays local
3. **Linux Native** - Optimized for Linux
4. **Lightweight** - Minimal resource overhead
5. **Modular** - Extensible architecture
6. **Open Source** - Community-driven

---

## Tech Stack

- **Backend**: FastAPI + Python
- **Frontend**: React + TypeScript + Tailwind
- **Desktop**: Tauri
- **Database**: SQLite + ChromaDB
- **AI Runtime**: Ollama (local inference)
